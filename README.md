# 👋 Sobre mí

Soy **Desarrollador Backend .NET** de **Paraná, Entre Ríos, Argentina**.

Cuento con más de **2 años de experiencia** desarrollando soluciones empresariales y bancarias utilizando **C#, ASP.NET Core, Entity Framework Core y SQL Server**.

Me apasiona desarrollar aplicaciones backend escalables, crear APIs REST, optimizar bases de datos y participar en la modernización de sistemas legacy.

Actualmente continúo capacitándome en:

- ☁️ Tecnologías Cloud
- 🏗️ Arquitectura de Software
- ✅ Clean Architecture y buenas prácticas de desarrollo

Siempre busco nuevos desafíos que me permitan seguir creciendo profesionalmente y aportar valor a los proyectos en los que participo.
